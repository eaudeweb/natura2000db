Aplicația Baza de date Natura 2000 este un catalog al siturilor Natura 2000 din România, disponibil liber online.

  • Siturile pot fi filtrate după cuvinte cheie, regiune administrativă, clase de habitat Corine și clasificare ca arie protejată.
  • Rezultatele sunt afișate suprapuse pe o hartă interactivă.
  • Se pot calcula statistici despre suprafețe totale în funcție de filtrări.

Aplicația este dezvoltată de către Agenția Națională de Protecție a Mediului. Codul sursă este disponibil pe GitHub iar baza de date poate fi downloadată în format JSON. Site-ul este găzduit de către Agenția Europeană de Mediu. {% endblock %}
